# AERIS — The Agentic Workspace Engine

AERIS ist eine hochperformante TUI-Anwendung (Terminal User Interface), die die Konzepte von **Antigravity**, **Cursor** und **Windsurf** in einer einheitlichen, masselosen Kommandozentrale vereint.

> Status im aktuellen Basics-Track: **PAUSED (Analyse-Modus)**.

## Kernphilosophie
AERIS ist kein Editor-Fork. Es ist ein **Workspace-Orchestrator**, der existierende Werkzeuge (Terminals, Editoren, KI-Modelle) koordiniert, um einen nahtlosen, agenten-gesteuerten Workflow zu ermöglichen.

## Hauptmerkmale
- **Agent Swarm Manager**: Parallele Ausführung und Überwachung von Agenten in dedizierten Terminal-Panes (Antigravity-Style).
- **Dynamic IDE Manager**: Flexible Auswahl von Terminal-Layouts (Grids, Columns, Custom) beim Projektstart.
- **Cascade-Flow**: Echtzeit-Awareness für Terminal-Outputs zur sofortigen Fehlerbehebung (Windsurf-Style).
- **OAuth Identity Bridge**: Sicherer, konsistenter Zugriff auf die Gemini-API (basierend auf der WANDA-Architektur).
- **Skill System**: Kompatibilität mit über 900+ Antigravity-Skills.
