# THE CANVAS — DAS UNIVERSAL UI-DESIGN SYSTEM

## 1. Das Konzept
**THE CANVAS** ist ein projektübergreifender UI-Baukasten, der es ermöglicht, komplexe Benutzeroberflächen (TUI oder GUI) visuell in Figma zu gestalten und direkt in funktionalen Code für AERIS oder zukünftige Projekte zu übersetzen.

## 2. Der Workflow (Elementor-Style)
1. **Design in Figma:** Jannis gestaltet das UI visuell mit vordefinierten "Canvas-Blöcken" (Menüs, Terminals, Status-Bars).
2. **Bridge-Interpreter:** Ein AERIS-Modul liest das Figma-Design via API aus.
3. **Live-Sync:** Design-Änderungen in Figma werden (nach Freigabe) sofort in das laufende Programm reflektiert, ohne dass die Logik neu geschrieben werden muss.

## 3. Vorteile
- **Visuelle Freiheit:** Jannis sieht sofort das fertige Bild, bevor eine Zeile Logik-Code steht.
- **Baukasten-Prinzip:** Schnelle Anpassungen von Layouts, Farben und Menüs direkt im Designer.
- **Nachhaltigkeit:** Einmal in "The Canvas" definiert, kann das UI-System für JEDES neue Projekt wiederverwendet werden.
- **Fehlerminimierung:** Das Design steht fest und ist valide, bevor die KI-Agenten darauf losgelassen werden.
